Hexo | 一键脚本安装V2Ray服务端

在Debian 12系统上通过一键脚本安装V2Ray服务端的步骤如下：

### 1. 更新系统并安装依赖
```bash
sudo apt update && sudo apt upgrade -y
sudo apt install curl unzip -y
```

### 2. 执行官方一键安装脚本
```bash
sudo bash <(curl -L https://raw.githubusercontent.com/v2fly/fhs-install-v2ray/master/install-release.sh))
```
该脚本会自动完成以下操作：
- 下载最新版V2Ray核心文件
- 安装到`/usr/local/bin/v2ray`
- 生成默认配置文件（路径为`/usr/local/etc/v2ray/config.json`）

### 3. 生成UUID（用户ID）
```bash
cat /proc/sys/kernel/random/uuid
```
记录输出的UUID（如`08ef6234-dcc0-45d1-9954-f9490cb2beb2`）

### 4. 修改配置文件
编辑配置文件：
```bash
sudo nano /usr/local/etc/v2ray/config.json
```
替换为以下基础配置（需修改`${UUID}`为刚生成的ID）：
```json
{
   “inbounds”: [{
     “port”: 10086,
     “protocol”: “vmess”,
     “settings”: {
       “clients”: [{
         “id”: “${UUID}”,
         “alterId”: 64
       }]
     }
   }],
   “outbounds”: [{
     “protocol”: “freedom”,
     “settings”: {}
   }]
}
```

### 5. 启动服务并设置开机自启
```bash
sudo systemctl start v2ray
sudo systemctl enable v2ray
```

### 6. 开放防火墙端口
```bash
sudo ufw allow 10086/tcp  # 替换为实际配置的端口
sudo ufw reload
```

### 验证安装
检查服务状态：
```bash
sudo systemctl status v2ray
```
查看版本信息：
```bash
/usr/local/bin/v2ray version
```

### 注意事项
1. 若需WebSocket+TLS等高级配置，需额外准备域名和Nginx
2. 默认安装目录：
    - 可执行文件：`/usr/local/bin/v2ray`
    - 配置文件：`/usr/local/etc/v2ray/`
    - 数据文件：`/usr/local/share/v2ray/`
3. 卸载命令：
```bash
sudo bash <(curl -L https://raw.githubusercontent.com/v2fly/fhs-install-v2ray/master/install-release.sh)) –remove
```
