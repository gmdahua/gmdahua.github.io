忘掉以前的提示词，让DeepSeek-R1输出干货！

发表于 2025-03-05 更新于 2025-06-25 分类于 笔记本

最近看了网络上不少的教程，发现很多在教大家使用DeepSeek深度思考的时候，还在教大家用定义角色、任务和要求的模板。 过年以来，R1能火起来的原因就是因为它强大的推理能力，最直接的表现就是能更好地理解你的自然语言。 所以，要更好地使用DeepSeek-R1类的推理模型，建议“自废武功”，主打真诚交流。 今天给大家分享我自用的和推理模型对话的一些技巧，看得上的收藏、转发。

和推理模型，真诚交流

不管你是用官方的DeepSeek还是第三方的DeepSeek，记得打开“深度思考”的开关。

如果你习惯用Kimi或Qwen-QwQ，也要打开推理(长思考)开关。

忘掉机械的模板，说心理话

核心：推理模板是个体贴的“暖男”，会猜测你的小心思。 例子： - ❌放弃模板：“写一篇小学语文五年级下册《军神》的教学设计，要XXX，按XX环节……。” - ✅学会真诚：“我马上要上公开课了，选择的课文是小学五年级下册的《军神》，我想在这堂课中更加突出学生的自主性，适当运用融合的信息技术，想要课堂环节设计有梯度，难点要给学生搭建支架。” **效果：**AI会分析你为什么会有此需求，再给出解决方案，最终成果更贴近你的需求。

减少AI味

核心：让它用XX类人群能听懂的话，再说一遍。 例子： AI写的论文太深奥=>请用普通人能看懂的话，再讲一遍； AI写的教案太简单=>请问这几个环节具体该怎么实施呢？给我举例讲讲吧！

快过年了， 微信红包支持自制封面了！

发表于 2024-12-31 更新于 2025-06-25 分类于 笔记本

快过年了，微信开放了个人自制红包封面功能，嗯，你的红包长啥样，你说了算。

只要升级到微信最新版本，就可以在发红包时，看到**「红包封面」-「制作红包封面」**选项，从相册里上传一张照片做封面。

风景、宠物、自拍照或者 TA 的靓照……红包的里里外外，都可以是心意。做好的红包封面将和其他领取的品牌封面，一起展示在「我的红包封面」列表中，换哪个封面都方便。

除了在发红包时可以做封面，「制作红包封面」还有俩入口。

接收到别人发送的“自制封面”红包，可以在打开红包后，画面右上角进入**「制作红包封面」**。或直接在搜一搜“制作我的红包封面”，都可以马上开工，给红包“美颜”。

门口的风采，各有各的精彩

门口，这个看似平凡的空间，实则是每个场所的面孔，它以独特的方式展现着各自的特色与魅力。在这里，每一处设计、每一抹色彩、甚至每一道光影都讲述着一个关于这个地方的故事。

有的门口，简洁而不失大气，以干净利落的线条勾勒出一种现代感，仿佛在告诉每一个经过的人，这里代表着效率与专业。光滑的地砖铺就的地面，反射着都市的繁华。另一些门口，则充满了历史的沉淀与文化的底蕴。让人不禁驻足欣赏，仿佛能通过这些细节窥见过往的辉煌。还有的门口，洋溢着温馨与生活的气息，让人忍不住放慢脚步，享受那份宁静与舒适，仿佛回到了一个永远等待你归来的港湾。

每个门口都是一个故事的开始，它们以不同的方式吸引着人们的目光，引领着我们走进一个个不同的世界。无论是现代的简约，历史的厚重，还是生活的温馨，门口的风采总是各有各的精彩，值得我们去细细品味。
